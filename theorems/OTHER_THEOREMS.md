# Other theorems from the campaign

Beyond the eight key theorems that carry the main proof, the campaign produced a number of further results — some structural stepping-stones, some independently interesting. They are listed here for completeness; each links to its standalone write-up. (These are provided in Markdown only.)

| Theorem | One line |
|---|---|
| Duality Theorem | the degreewise Gorenstein socle pairing underlying the top-zone reduction |
| Glue Theorem | how the sheet-local data reassembles into a global element |
| Tower-Collapse Theorem | the mechanism by which level $v+1$ is an exact $\times 3$ copy of level $v$ |
| Three-Cut-Bone Theorem | a reduction of the collar obstruction to three cuts |
| Universal SOP Theorem | the universal system-of-parameters structure of the arrangement |
| Field-Equation Bone Theorem | the field-equation constraints at the collar |
| Split Spectrum Lemma | the totally-split linear spectrum of the even sextic's splitting algebra |
| Gröbner–Frobenius Glue Theorem | the gap/Hilbert–Kunz relation used as an independent cross-check |
| Normalization Theorem | the normalization used in the frame construction |
| Per-Edge Determinant Theorem | the 15-determinant transversality certificate of the frame |
