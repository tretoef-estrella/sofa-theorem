# Other results from the development

The eight key theorems (see the [main README](https://github.com/tretoef-estrella/sofa-theorem/blob/main/README.md)) carry the main proof and are provided in full, in Markdown and PDF. Beyond them, the development produced a number of further results — structural stepping-stones used to assemble the proof, and a few lemmas of independent interest. They are catalogued here for completeness.

These are **supporting machinery**: each is used inside the proof of one of the key theorems or the main paper, and each is stated and used there. They are listed so that a reader following a specific step knows the result has a name and a self-contained argument.

| Result | Role in the proof |
|---|---|
| **Duality Theorem** | the degreewise Gorenstein socle pairing underlying the top-zone reduction (used in Proposition 1.4 and Theorem 3.1) |
| **Per-Edge Determinant Theorem** | the fifteen-determinant transversality certificate establishing that the frame $\ell_1,\ell_2,\ell_3$ is transverse on every sheet (used in Proposition 1.4) |
| **Normalization Theorem** | the normalization fixing the frame construction |
| **Glue Theorem** | how sheet-local data reassembles into a global element |
| **Tower-Collapse Theorem** | the mechanism by which level $v+1$ is an exact $\times 3$ copy of level $v$ |
| **Three-Cut-Bone Theorem** | a reduction of the collar obstruction to three cuts |
| **Universal SOP Theorem** | the system-of-parameters structure of the arrangement |
| **Field-Equation Bone Theorem** | the field-equation constraints at the collar |
| **Split Spectrum Lemma** | the totally-split linear spectrum of the even sextic's splitting algebra |
| **Gröbner–Frobenius Glue Theorem** | the gap / Hilbert–Kunz relation, retained as an independent cross-check of the main computation |

The full write-ups of these supporting results are available on request to anyone undertaking a detailed review — write to **tretoef@gmail.com**. The main proof and its eight key theorems are self-contained without them.
